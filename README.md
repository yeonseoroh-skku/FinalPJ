# FinalPJ
👗 Core-Trend-Dashboard

An interactive, data-driven web environment designed to isolate, structure, and visualize the meteoric rise of Gen-Z digital **"-core" aesthetics** and subcultures. Built within a Python programming framework and optimized for lightweight web delivery.

---

## 📌 Project Overview

The contemporary fashion and beauty ecosystems are no longer governed solely by elite heritage houses. Instead, they are rapidly reshaped by hyper-specific, algorithmically driven micro-trends born on social platforms like TikTok, Instagram, and Pinterest. 

**Core Trend Dashboard** serves as both an academic exploration of digital consumer sociology and a practical blueprint for modern retail trend analysis. By converting high-velocity internet subcultures into organized visual profiles, this platform eliminates structural informational noise and maps direct commercial pathways to market leaders.

---

## 🎯 Key Features & Analytical Scope

While the digital landscape fragments into hundreds of fleeting micro-fads, this platform strategically isolates **six definitive fashion-core aesthetics** currently dominating Gen-Z consumer behavior and retail search surges:

* **✨ Meta-Aesthetic Home Engine**
  * Contextualizes the overarching linguistic framework of the "-core" suffix.
  * Explores the meta-emergence of the "Corecore" cultural phenomenon.

* **🌲 Gorpcore Module**
  * Analyzes the fusion of high-performance outdoor utilitarian gear with urban streetwear.
  * Driven by themes of comfort and climate readiness.

* **🩰 Balletcore Module**
  * Tracks the transition of classical dancewear (leotards, tulle skirts) into everyday style.
  * Focuses on romantic, hyper-feminine elevated athleisure.

* **🤓 GeekChic Module**
  * Decodes the cultural recontextualization of traditionally "scholastic" or "nerdy" apparel.
  * Highlights the rise of luxury "Librarian-Core" statements.

* **💖 Barbiecore Module**
  * Examines early-2000s nostalgia and dopamine dressing.
  * Showcases unapologetic, maximalist pink glamour within fashion and cosmetics.

* **🤠 Cowboycore Module**
  * Maps the mainstreaming of authentic Western heritage wear (denim, fringe, leather).
  * Backed by high-profile pop-culture endorsements.

* **👑 Royalcore Module**
  * Explores historical escapism and regal European opulence.
  * Popularized by digital period-drama fandoms.

## 🛠️ Technical Architecture

### 1. Presentation Layer (UI/UX)
* **Single-Page Application:** Engineered using modern semantic HTML5 and vanilla JavaScript tab-switching logic to ensure instantaneous rendering and zero-refresh page transitions.
* **Sleek Aesthetic Design:** Features a responsive layout styled with a soft pastel backdrop, sophisticated typography contrast, and elegant glassmorphism module containers.
* **Interactive Components:** Implements smooth-folding custom accordion engines for structured brand analytics and detailed aesthetic anatomy data.

### 2. Technology Stack & Infrastructure

| Component | Technology | Description |
| :--- | :--- | :--- |
| **Frontend** | HTML5, CSS3, JavaScript (ES6+) | Advanced Flexbox, Backdrop Filters, and Vanilla JS UI logic. |
| **Deployment** | GitHub Pages | Serverless, zero-install, and fully responsive web hosting. |

## 👥 Target Audience & Utility

* **Academic Evaluation:** Proves structural data-mapping capabilities and digital subculture categorization methodologies within the context of big data and creative trend tracking.
* **Industry Application (B2B):** Assists boutique fashion brand marketers and indie retail buyers in optimizing inventory alignments based on ascending internet aesthetics.
* **Consumer Insight:** Translates abstract internet "vibes" into tangible, structured variables—explicitly listing the authoritative brands and core elements defining each movement.
